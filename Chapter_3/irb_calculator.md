#Using irb to do some calculations.

### Whole number Calculations
```
2.3.1 :001 > 3 + 2
 => 5
2.3.1 :002 > 100 % 3
 => 1
2.3.1 :003 > 9758398584 % 7
 => 4
2.3.1 :004 > 9758398584 / 7
 => 1394056940
2.3.1 :005 > 8*4/3+6 > 2-1*80+3%4
 => true
 ```
 
 ### Floating point Calculations
```
2.3.1 :006 > 21.0 / 4.0
 => 5.25
2.3.1 :007 > 21.0 % 4.0
 => 1.0
2.3.1 :008 > 4.0 * 3.2
 => 12.8
2.3.1 :009 > 19.7 - 4.5
 => 15.2
 ```
